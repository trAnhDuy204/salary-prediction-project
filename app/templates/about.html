Giới Thiệu Dự Án

Hệ Thống Dự Đoán Mức Lương Bằng Machine Learning

Tổng Quan Dự Án

Dự án Salary Prediction System là một ứng dụng web sử dụng công nghệ Machine Learning để dự đoán mức lương phù hợp dựa trên các yếu tố như kinh nghiệm làm việc, kỹ năng chuyên môn, vị trí công việc và khu vực địa lý.

Với hơn 85,000 tin tuyển dụng từ thị trường lao động Việt Nam tính đến ngày 22 tháng 4 năm 2025, hệ thống được huấn luyện để cung cấp những dự đoán chính xác và hữu ích cho cả người tìm việc lẫn nhà tuyển dụng.

Tính Năng Chính

Machine Learning Tiên Tiến

Sử dụng thuật toán Random Forest Regressor được tối ưu hóa với:

  • Hyperparameter tuning với GridSearchCV
  • Cross-validation 5-fold
  • Feature importance analysis
  • Độ chính xác R² > 0.80

Dữ Liệu Thực Tế

Dựa trên bộ dữ liệu lớn từ thị trường Việt Nam:

  • 85,000+ tin tuyển dụng
  • 10+ thành phố lớn
  • 20+ lĩnh vực nghề nghiệp
  • Cập nhật đến tháng 4/2025

Phân Tích Chi Tiết

Kết quả dự đoán bao gồm:

  • Mức lương dự đoán chính xác
  • Khoảng tin cậy (confidence interval)
  • Phân loại mức lương (thấp/trung bình/cao)
  • Thông tin về độ chính xác mô hình

RESTful API

Dễ dàng tích hợp vào hệ thống của bạn:

  • API endpoint đơn giản
  • JSON request/response
  • CORS enabled
  • Tài liệu API đầy đủ
